## 4. tool_demo.py

**功能说明**：演示单个工具调用的完整流程（获取当前时间）。

**核心逻辑**：
- 定义 `get_current_time` 工具函数，并按 OpenAI 工具规范声明其 schema。
- **第一步**：调用模型，传入用户问题（"现在几点了？"）和工具列表。
- 模型返回 `tool_calls` 请求调用 `get_current_time`。
- **第二步**：执行对应的 Python 函数，获得时间字符串。
- **第三步**：将工具结果以 `role: "tool"` 的形式追加到消息列表中。
- **第四步**：再次调用模型，模型这次能看到工具执行结果并给出自然语言回答。
